This is a simple frontend / html framework for building mobile friendly very simple pages.

Idea is to simply structure the content on the file system.

Plus:

  • Also there is some routing like for 'webaddress.com/someRoute.
    (Only works when delivered by a server / not from file system due to CORS)

  • An easy image caroussel.

  • Tracking of the visitors. (Available as Java Server Feature)

Be cautious!
Regular tags used in the body part must always be opening and closing though their content is empty!

<tag></tag>
instead of
</tag>

This is due to javascript handling and parsing the DOM.
Not following that hint may result in confusing element order when sub parts are dynamically loaded.
I did not find a solution for that yet.
